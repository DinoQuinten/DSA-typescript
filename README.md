# 🧠 DSA Mastery — The Only Reference You'll Ever Need

> A comprehensive, no-fluff DSA course in TypeScript. Built for LeetCode prep and beyond.

## 🗺️ How to Use This Guide

1. **Read in order** the first time — each chapter builds on the previous
2. **Use as reference** later — jump to any folder when you're stuck on a LeetCode problem
3. **Run the code** — every folder has a named `.ts` file you can execute with `bun run`

## ⚡ Quick Setup

```bash
bun install
# Run any chapter's code:
bun run typescript-fundamentals/typescript-basics.ts
bun run 00-big-o-notation/big-o.ts
bun run 05-trees/trees.ts
```

## 🎯 LeetCode Strategy After This Course

Once you've read through everything:

1. **Easy problems first** — build confidence and pattern recognition
2. **One topic at a time** — do 5-10 problems per topic before moving on
3. **Time yourself** — 20 min for Easy, 40 min for Medium, 60 min for Hard
4. **Review, don't just solve** — come back to this guide to understand WHY a pattern works

Every chapter follows this exact format:

1. 🌍 **Real-World Analogy** — a mental model that sticks forever

5. ⏱️ **Complexity Analysis** — time & space for every operation
6. 🎯 **LeetCode Patterns** — when to reach for this tool
7. ⚠️ **Common Pitfalls** — mistakes to avoid
8. 🔑 **Key Takeaways** — rapid-fire summary for revision
9. 📋 **Practice Problems** — curated LeetCode problems by difficulty
