# Visual DSA Lab

Interactive Svelte pages for Dijkstra, graphs, recursion, backtracking, and dynamic programming.

## Run

```bash
bun install
bun run dev
```

If `bunx create-vite` fails due to temp directory permissions, set:

```bash
$env:BUN_TMPDIR="D:\Prasanna-application-tools\learning dsa\.bun-tmp"
$env:BUN_INSTALL="D:\Prasanna-application-tools\learning dsa\.bun-install"
```
